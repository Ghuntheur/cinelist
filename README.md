# CINELIST

### Installation

```bash
npm i
npm run dev

# build
npm run build

# Créer un fichier .env avec une clé api TMDB
# https://developers.themoviedb.org/3/getting-started
```

### Technos

- React
- Typescript
- Redux (toolkit)
- Dayjs
- TMDB api
